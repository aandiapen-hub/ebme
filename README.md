# EBME

A Dockerized Django application for managing medical equipment inventories and maintenance records.

## Status
🚧 Work in progress. Documentation is coming.

## Purpose
This project focuses on asset and maintenance tracking only.
It does not process patient data and is not a medical device.

## Tech Stack
- Django
- Docker / Docker Compose
- PostgreSQL (or whatever you use)
